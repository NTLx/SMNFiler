As far as the current settings are concerned, the font of the report generated by default is Arial and presented in the form of a PDF file. If you need to adjust the font, you can turn on the switch button `MiSanså­—ä½“`. If you need to adjust the generated report file to be presented in the form of a web page, you can turn on the `ç”ŸæˆHTMLæ–‡ä»¶` switch button.
